# Delay Strategy

* Supported Delay Strategies
* * Fixed
* * Linear
* * Exponential
* Decorators
* * Jitter
* * Correlated Jitter

Create a Based Delay class and it should be extended by the linear fixed and exponential delay classes

It should return a delay number in ms

Now this returned delay will be modified by decorators can be normal or correlated jitter

The final should be a result of these 2

Delay Strategy:
* Linear
* Exponential
* Fixed

Jitter Strategy:
* Full
* Equal
* decorellated
